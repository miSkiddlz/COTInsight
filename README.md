**COTInsight** is an interactive dashboard that automatically reads weekly CFTC Commitments of Traders (COT) Reports and visualizes market positions.  
Users can quickly explore Long vs Short positions, Net Positions, and Trader Types (Commercial, Speculators, Retail) across major futures markets.

Visit: https://cotinsight.onrender.com

Features

- Automatic download and processing of COT Reports
- Interactive charts by Asset and Trader Type
- Net Position calculation and visualization
- Easy-to-use, browser-based interface
- Fully open source and extendable
